카드 이미지는 이 폴더에 넣으면 됩니다.

우선 인식하는 파일명:
- animal1.png
- animal2.png
- ...
- animal40.png

추가로 아래 형식도 같이 인식합니다.
- pair01.png
- pair02.png
- ...
- pair40.png

지원 확장자:
- png
- jpg
- webp
- jpeg

현재 게임에서 고를 수 있는 카드 장수:
- 16장
- 20장
- 24장
- 30장
- 36장
- 40장

총 장수의 절반만큼 서로 다른 카드 이미지가 필요합니다.
예: 40장 게임이면 이미지 20종이 필요합니다.
